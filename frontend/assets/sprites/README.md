# Sprite Assets

Add pixel art sprites here in later iterations.

## Expected Format
- PNG with transparency
- Consistent sprite dimensions per character type
- Character facing direction: 4 frames (up, down, left, right)
- Animation frames for walking: 2-4 frames per direction

## Naming Convention
- player.png
- einstein.png
- tesla.png
- abdulkalam.png
- gandhi.png
